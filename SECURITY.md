# Reporting Security Issues

If you discover a security vulnerability in OpenGame, please report it privately by opening a [GitHub Security Advisory](https://github.com/leigest519/OpenGame/security/advisories/new) on this repository, or by emailing the maintainers directly. Please do **not** open a public issue for security reports.

We will triage the report, work with you on a fix, and coordinate disclosure once a patch is available.

Thank you for helping keep OpenGame secure.
